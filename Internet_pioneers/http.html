HTTP

The HTTP or the hypertext Transfer Protocol is a system of requests between the client and server.

It is part of the Internet protocol suite and defines commands and services used for transferring webpage data.
HTTP uses a server- client model. A client, for instance, may be a home computer, laptop, or mobile device.
The HTTP server is generally a web host running web server software

How it works:

  • GET method is used to request an entire resource.
  • HEAD method is for a certain resource without the contents of the body.
  • POST method updates an existing online resource by adding content, messages, or data to a new page.
  • PUT method generates a new URI or changes an existing web resource.
  • DELETE method removes a certain resource.
  • TRACE method displays any modifications or additions of an online resource for users to see.
  • OPTIONS method displays a list of possible HTTP methods for a given URL.
  • CONNECT method establishes a transparent TCP/IP tunnel from the request connection.
  • PATCH method modifies a web resource.
